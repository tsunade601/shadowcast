# ShadowCast

**A decentralized, client-side P2P mesh network engine** that mirrors and streams static web assets directly browser-to-browser using WebRTC, drastically reducing server bandwidth and hosting costs.

## Key Features
- **P2P Mesh Network**: Browser-to-browser connections via WebRTC
- **Asset Mirroring & Streaming**: Efficient distribution of static content
- **Significant Bandwidth Savings**: Offload server costs to the client mesh
- **Low-Latency Optimization**: Real-time performance focus
- **WebSocket Fallback**: Ensures reliable connectivity

## Technologies
- WebRTC
- WebSockets
- Modern JavaScript / TypeScript
